## 🔗 Get Pair Code (No QR Needed)

Run:

```bash
node pair.js
Example: 919876543210
ABCD-1234
``` 
  

### The Hard Method
💻 Termux / Local Setup
```js
$ apt update && apt upgrade
$ pkg install nodejs git -y

$ git clone https://github.com/MrAxenzo/AxA-BOT
$ cd AxA-BOT

$ npm install

# Get Pair Code
$ node pair.js

# After linking
$ node bot.js
```
